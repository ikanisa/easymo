# ⚠️ DEPRECATED: admin-app-v2

**This application has been deprecated in favor of `admin-app`.**

## Migration Status

As of 2025-11-27, all unique features from admin-app-v2 have been migrated to admin-app.

## Why Deprecated?

- **admin-app** has Tauri desktop support
- **admin-app** has Sentry integration
- **admin-app** uses shared packages (@easymo/commons, @va/shared)
- **admin-app** has React Query persistence
- Maintaining two admin apps creates confusion and duplicate effort

## Removal Timeline

- **2025-12-01**: Remove from CI/CD pipelines
- **2025-12-15**: Archive directory
- **2026-01-01**: Delete from repository

## Questions?

Contact the frontend team before the removal date if you have dependencies on this app.
