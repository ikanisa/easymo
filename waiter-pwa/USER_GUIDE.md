## Troubleshooting

### App Won't Load
1. Check internet connection
2. Clear browser cache
3. Update to latest browser version
4. Reinstall the PWA

### Chat Not Responding
1. Check internet connection
2. Refresh the page
3. Start a new conversation
4. Clear app data and retry

### Items Not Adding to Cart
1. Ensure item is available
2. Check if restaurant is open
3. Refresh the page
4. Try again

### Payment Failed
1. Check card/account balance
2. Verify payment details
3. Try another payment method
4. Contact your bank if issue persists

### Can't Make Reservation
1. Check if restaurant accepts reservations
2. Verify date/time availability
3. Try alternative times
4. Call restaurant directly

### Order Not Showing
1. Go to Orders page
2. Pull to refresh
3. Check order confirmation email
4. Contact support with order number

### App Running Slow
1. Close other apps/tabs
2. Clear browser cache
3. Restart your device
4. Check for app updates
