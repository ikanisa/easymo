# @easymo/media-utils

Audio and media processing utilities for the EasyMO platform.

## Installation

```json
{
  "dependencies": {
    "@easymo/media-utils": "workspace:*"
  }
}
```

## Usage

```typescript
import {
  base64ToFloat32Array,
  float32ToPCM16,
  createBlob,
  decodeAudioData
} from '@easymo/media-utils';
```
